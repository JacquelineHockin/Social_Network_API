# Social_Network_API


# Description

This is social network API web application using a configured working Express API that interacts with a MongoDB database and Mongoose ODM, allowing users to share their thoughts, react to friendsâ€™ thoughts, and create a friend list. These thoughts, thought reactions, and friend lists can be updated and deleted.

## Installation

Run "npm init" to set up all existing npm packages

"npm install" to install necessary dependencies 

Check that MongoDB and Express are installed on your computer


## Usage 

npm start

With Nodemon:
npm run watch
